🌤️ Weather App

A simple, responsive weather app built with HTML, CSS, and JavaScript that shows real-time weather and a 5-day forecast for any city.

Features

1.Search weather by city name
2.Get weather for your current location (Geolocation)
3.5-day weather forecast
4.Dark / Light mode toggle
5.Celsius / Fahrenheit unit toggle
6.Recent searches (last 5 cities)
7.Dynamic background based on weather conditions
8.Loading spinner while fetching data

Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- [OpenWeatherMap API](https://openweathermap.org/api)

How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Search for a city or click the 📍 button to use your current location

Setup

To run this project with your own API key:

1. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Open `app.js`
3. Replace the `apiKey` variable with your own key:
```js
const apiKey = "YOUR_API_KEY_HERE";
```
